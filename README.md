# 🎵 MAXIMUSIKAI STUDIO

Välkommen till **MAXIMUSIKAI** – den ultimata AI-drivna studion för musik och video! 🚀

### ✨ Funktioner
- **🪄 TOTAL MAGI:** Skapa en komplett musikvideo från en enda textrad.
- **🎬 REGISSÖREN:** Animera dina egna bilder med AI.
- **🎧 BARA MUSIK:** Generera unika soundtracks och beats.
- **🌐 COMMUNITY:** Dela dina skapelser med andra användare (Supabase).
- **🎨 NEON DESIGN:** Fullmatad med cyberpunk-estetik och ljusa, tydliga menyer.

### 🛠️ Teknik
- **Frontend:** Streamlit
- **AI-Motorer:** Replicate (Flux, Llama-2, Minimax, Musicgen)
- **Video-mix:** MoviePy
- **Databas:** Supabase

### 🚀 Kom igång
1. Lägg till din `REPLICATE_API_TOKEN` i Streamlit Secrets.
2. Lägg till `SUPABASE_URL` och `SUPABASE_KEY` för community-funktioner.
3. Kör `streamlit run app.py`!

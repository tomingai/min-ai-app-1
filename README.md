# 🎬 AI Video & Musik Studio

Detta är en kraftfull AI-app byggd med **Streamlit** och **Replicate API**. Den tillåter användare att förvandla en enda bild till en rörlig video med matchande AI-genererad musik.

## ✨ Funktioner
- **Bild till Video:** Använder *Stable Video Diffusion* för att animera dina bilder.
- **AI-Musik:** Skapar unika soundtracks med *Meta MusicGen*.
- **Enkelt Interface:** Byggt helt i Python för en smidig användarupplevelse.

## 🚀 Kom igång
1. Gå till [Replicate](https://replicate.com) och hämta din API-nyckel.
2. Ladda upp en bild i appen.
3. Tryck på generera och vänta på magin!

## 🛠 Teknik
- [Streamlit](https://streamlit.io)
- [Replicate API](https://replicate.com)
- Python
